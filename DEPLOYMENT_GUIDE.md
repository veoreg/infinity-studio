# Руководство по деплою (Manual Deployment Guide)

Этот документ поможет вам самостоятельно задеплоить проект, если автоматика не сработала или нужно внести изменения вручную.

## 1. Где искать бэкап?
Свежий бэкап текущей (рабочей) версии интерфейса находится здесь:
`e:\AI_girl_flux_dev\APP_VID\ai-generation-front\src\snapshots\2026-01-23_FINAL`

## 2. Подготовка к деплою
Чтобы задеплоить проект на Netlify и Vercel, вам нужно отправить код в GitHub.

### Шаг 2.1: Сборка проекта (Build)
Перед деплоем желательно проверить, что проект собирается без ошибок:
1. Откройте терминал в папке `ai-generation-front`.
2. Введите команду:
   ```cmd
   npm run build
   ```
Если в конце написано `✓ built in ...s`, значит всё в порядке.

### Шаг 2.2: Отправка в GitHub (Git Push)
GitHub — это «сердце» деплоя. Как только код попадает туда, Netlify и Vercel подхватывают его автоматически.

Выполните эти команды по очереди:
1. **Добавить все изменения**:
   ```cmd
   git add .
   ```
2. **Создать сохранение (Commit)**:
   ```cmd
   git commit -m "update: stable version for demo"
   ```
3. **Отправить на сервер (Push)**:
   ```cmd
   git push origin main
   ```

## 3. Как проверить статус деплоя
- **Netlify**: Перейдите в панель управления Netlify. В разделе «Deploys» вы увидите текущую сборку. Она должна быть в статусе `Ready`.
- **Vercel**: Аналогично в панели Vercel, вкладка «Deployments».
